# Cavalry
 Unity Project
# 프로젝트 개요

- 말을 타고 전장을 달리며 활을 쏘며 적들을 섬멸하는 게임

# 게임 개요

- 플레이어는 멈추지 않고 끊임없이 이동해야 합니다. 오로지 이동 방향만 정할 수 있습니다.
- 활 시위를 끝까지 당기면 훨씬 강력한 공격을 할 수 있습니다.
- 화살을 발사하여 적의 투사체를 파괴할 수 있습니다.
- 환도를 이용하여 적의 투사체를 방어하거나 튕겨낼 수 있습니다.

# 조작키

**기본 조작**

- WASD: 이동 방향 설정
- 우클릭 누름: 활 시위 당기기
- 우클릭 떼기: 화살 발사
    - 누른 시간에 비례하여 강하게 발사(최대 1.5초)
    - 최대치로 발사할 시 관통 효과가 부여되며 데미지가 2.5배 증가

**스킬 활성화 시 사용 가능**

- 좌클릭: 환도 사용
    - 가드와 반사 중 하나를 선택하여 사용할 수 있음
    - 가드: 1초 동안 적의 투사체를 방어
    - 반사: 칼을 휘둘러 적의 투사체를 반사
- 스페이스바: 대쉬
    - 달리는 방향으로 짧은 거리를 빠르게 이동
- 쉬프트: 더미 설치
    - 마우스가 가리키는 방향으로 더미를 설치
    - 더미는 일정 시간 동안 주변 몬스터의 공격 타겟이 된다.
